readin functions¶

Functions for locating, opening, or reading files.

List by function¶

Creating a directory¶

checkdir - create a directory if it does not exist.

Encoding (unicode)¶

unicodify - determine character encoding and convert to unicode.

Getting file information¶

file_signature - get a file’s format (i.e., xlsx, xml, …).

get_absolute_path - return absolute url of web-base file based on domain name and relative url.

get_fnames - get list of file names in a directory.

most_recent - file the most recently modified file in a directory.

GitHub files¶

check_last_load - check for last file loaded on GitHub.

scan_csv_list - scan files on GitHub and return names and URLs for all CSVs.

Reading file¶

read_yaml - scan a yaml file and return user specified data block.

Regex¶

date_in_str - extract a date pattern from a string.
